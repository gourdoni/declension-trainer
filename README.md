## Usage

### Prerequisites

- JDK 21+
- Maven 3.9+

### Execution

The scripts below build a self-contained `jar` and start the server:

- UNIX: `./run.sh`
- Windows: `run.bat`

To do this manually on any platform:

```
mvn clean package
java -jar target/declension-trainer.jar
```
